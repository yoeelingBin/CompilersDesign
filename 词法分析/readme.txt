## 要求

利用flex编写一个词法分析器，能够
1. 提取出.sealpp文件中各种单词
2. 排除注释
3. 正确标定单词行号
4. 计算整个.sealpp文件中的行数，字数，字符数

单词包括：
- 关键词:fprintf while aafor if else continue break return
- 运算符:+ - * / % && & < <= > >=等
- 常量:整型、浮点、字符串、布尔
- 符号:函数名、变量名
- 其他符号：逗号、分号、括号等

对测试文件进行分析，需要完全一致
